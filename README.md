# Boso Karta Typing App
---
## Lets typing !

- 開発環境での起動方法(win、mac共通)
  - `git clone https://github.com/Shun712/boso-karuta.git`
  - `git pull origin main`
  - `npm install`
  - 環境変数ファイル ".env" を以下の通り ”/karuta_typing/" ディレクトリ配下に格納する (環境変数ファイル.envは、コラボレーターに展開を依頼してください)
  - `npm run dev`
